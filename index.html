Crestron Panel Help Guide

The following guides cover the use of the room control panel. Please use the menu to the left to review specific tasks.

Controls Overview

Below you will find a simulation of the room control panel. Click the plus (+) icons in order to review relevant information about each control.

If you are still unsure how to use the panel please contact us so that we may assist you.

Turn System On

Turning the system on is a simple task.

  1. Tap the screen to wake the system up.
  2. The screen will turn on and offer the room's display options. Tap the desired option.
Return to Top ^

Choose Source

You will now need to select a source to project or display on the room monitor.

Display Options

  • Conference - Run or join a Zoom/Skype for Business meeting.
  • Computer - Standard presentation delivery that is not recorded or part of a meeting.
  • Laptop/HDMI - Display a laptop connected to the console.
  • Doc. Camera - Display physical media on the screen/monitor using the document camera.
  • AirMedia - Connect AirMedia supported device for display.
  • Tie Line Panel - Contact Systems Engineering for assistance with this option.

Tap the desired source option.

Changing Source

If you select the wrong source by mistake you may simply tap the correct source of your choice.

Adjust Mic Volume

The microphone levels can be adjusted directly from the control panel.

  • Tap the up arrow under Mic Master Level to increase microphone volume.
  • Tap the down arrow to decrease microphone volume.
  • The level bar indicates current microphone volume from minimum to maximum.
Return to Top ^

Adjust Source Volume

The source (meeting, computer, etc) volume levels can be adjusted directly from the control panel.

  • Tap up arrow under CPU Volume to increase source volume.
  • Tap the down arrow to decrease source volume.
  • The level bar indicates current source volume from minimum to maximum.

Mute Microphones

Microphones can be muted/unmuted on the control panel. This allows you to have your microphone on and ready without your audio being projected to the room.

  • Tap ALL MICS UNMUTED to mute microphones. Note that the icon changes to indicate microphones are muted.
  • Tap ALL MICS MUTED to unmute microphones.

Note: At a glance you can tell if microphones are muted if the microphone icon has a slash through it.

Return to Top ^

Mute Projectors

Projectors can be muted/unmuted from the control panel. This allows you to control when the audience will see content on the projectors.

  • Tap Projector Mute to mute projectors. Note that the icon changes to indicate projectors are muted. Verify that the projectors are muted by looking at the screen(s) in the room.
  • Tap Projector Mute again to unmute projectors. Note that the flashing MUTED text is no longer visible.

Enable PowerPoint Notes

Some presenters are more comfortable using the notes section in PowerPoint. The notes can be displayed on both the confidence monitors and the podium computer.

  • Tap the PP NOTE IS OFF button to enable PowerPoint notes on the confidence monitor. Note that the text changes to read PP NOTES IS ON
    • The computer must be set to Extended Display mode. If you're unsure how to set this view the steps here.
  • Tap PP NOTES IS ON to disable PowerPoint notes.

The following guides assist with common tasks you will encounter on the room computer.

Zoom Meetings

Zoom is a simple and convenient way to host and join web-based meetings.

  1. Launch Zoom by double-clicking the app icon on the desktop. You may also launch Zoom directly from a meeting invitation.
  2. Click Join a Meeting if you are not joining from a meeting invite link.
  3. Enter the meeting ID for your meeting.
  4. Click Join with Computer Audio.

Skype for Business Meetings

Skype for Business is another option for hosting web-based meetings.

  1. Launch Skype for Business by double-clicking the app icon on the desktop. You may also launch Skype for Business directly from a meeting invitation.
  2. Log in with your NID and NID password.
  3. Click the calendar icon if you are not joining from a meeting invite link.
  4. Double-click on the meeting you wish to join/start from the list that appears.
  5. Select Use Skype for Business (full audio and video experience) on the Join Meeting Audio prompt screen.
  6. Click OK.

Your meeting is now launched and ready to begin.

Enable PowerPoint Notes

PowerPoint notes can be displayed on the confidence monitor(s) for presenters that wish to use them.

  1. Enable PowerPoint Notes on the control panel.
  2. Double-click the Extend icon on the desktop.
  3. If you do not see the Extend icon press the windows key and p at the same time then click Extend.

The PowerPoint notes section will now appear on the confidence monitor(s) in the room.

Save Documents

Classroom and meeting room computers have security software installed which will delete all files upon restart unless they are saved to a specific folder.

Note: It is a good idea to save presentations to this folder rather than the desktop or running directly from USB media.

  1. Download your presentation or insert your USB media into the computer.
  2. Double-click on the Presentations - Save Files Here folder.
  3. Click and drag your file into the folder.
  4. Double-click your file from the Presentations - Save Files Here folder to begin the presentation.

Use Pen Tool in PowerPoint

Using the pen tool to annotate PowerPoint slides is a great way to elaborate on slide content.

  1. Open the PowerPoint presentation.
  2. Start slide show mode.
  3. Press the control and P keys at the same time.
  4. Use the stylus connected to the monitor to draw on the screen.
Return to Top ^

Use Laser Pointer in PowerPoint

PowerPoint has a built-in laser pointer that is useful for highlighting important slide content.

  1. Open the PowerPoint presentation.
  2. Start slide show mode.
  3. Press the control and L keys at the same time.
  4. Use the mouse to move the laser dot and highlight content on the slide.

Note: This is preferable to a physical laser pointer because it will be included during playback of recorded sessions.

Start Panopto Recording

If a session needs to be recorded you can start a Panopto session.

  1. Right-click on the Panopto icon in the system tray at the bottom-right of the screen.
  2. Select Start New Recording.
  3. Click Yes on the prompt that appears. A red dot on the Panopto icon indicates recording has started.
  4. The recording time defaults to one hour. Right-click the Panopto icon and select Extend 5 minutes if you need additional time.
  5. Right-click the Panopto icon and select Stop or press F10 to stop the recording.

Note: Panopto recordings should normally be scheduled ahead of time through Ed Tech. If you start a recording please contact Ed Tech as soon as possible. If you are recording outside of standard business hours please text 407-308-8089 to request assistance.

Use Room Radio

Each room has a radio which can be used to call for assistance.

  1. Use the channel select knob to choose either ED Tech or IT.
  2. Press and hold the talk button. A beep will let you know that the channel is open.
  3. State the that you need assistance. Make sure to provide a room number.
  4. Release the talk button.

Note: The volume on the radio in the room will be turned down to avoid distractions during meetings and lectures. This means you will not hear our response but rest assured that someone is on the way. Feel free to repeat your call.
